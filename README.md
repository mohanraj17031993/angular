# angular

This workspace contains a minimal Angular v21 project scaffold.

Commands:

- Install dependencies: `npm install`
- Start dev server: `npm start` (uses `npx @angular/cli@21 serve`)
- Build: `npm run build`

Notes:

- The project specifies Angular `^21.0.0` packages in `package.json`.
- If you prefer to use a globally installed CLI, run `ng serve` after installing `@angular/cli`.

- The app uses Angular standalone components (no `NgModule`; bootstrapped with `bootstrapApplication`).
